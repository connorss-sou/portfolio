Group Addition Trigger Homepage

Author: Sean Connors

Description:

This pages allows you to see the affects the group addition trigger has when a new contributor is added to an existing group.



Justification:

This page is useful for users with high privalege such as administrators who may make changes in a group.

It would also be useful for when the eventual process of group movement would be automated, and this would allow for records of what is being done.


Please select a contributor and a group to move them to.

If you would like to remove the contributor from the group select the remove option. You can pick any of these options really, but you will encounter an error if the contributor is already in the group. In this case, you can just go back and try again. I can't give an example, as this would be repeated and would trigger the error, so feel free to try it. You will know it worked if the log appears and shows you all of the entries to the log.
