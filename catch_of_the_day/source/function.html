Find Likely Location

Author: Sean Connors

Description:

This page allows the user to view the likely location of where a selected fish may be caught. This is based which location has the most amount of catches for the selected fish.



Justification:

This page is useful for all contributors and users who may want to find a location to catch a certain fish.


A good example fish is steelhead or chinook salmon because they have each been caught multiple times at a couple locations. If you pick a steelhead, there are currently 8 steelhead catch records, and the location with the most of these is the Illinois River. For some fish, there are no catch records associated, so nothing will be returned, so just go back and choose another fish.
